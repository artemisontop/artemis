Current Version: V0.1

# Games Supported

* Super Power Training Simulator
* Arsenal (BETA)
* Strongest Punch Simulator
* Lifting Simulator
* Pet Simulator X (ALPHA)
* Bubble Gum Simulator
* Universal (BETA)
